# 🧮 Calculator Web Application

A simple and responsive calculator built using **HTML**, **CSS**, and **Vanilla JavaScript**. This project performs basic arithmetic operations through an interactive and user-friendly interface.

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear (C) button
- ⌫ Backspace functionality
- 📱 Responsive and clean UI
- ⚡ Instant calculation using JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 Project Structure

```
Calculator/
│
├── index.html      # Structure of the calculator
├── style.css       # Styling and layout
├── script.js       # Calculator logic
└── README.md
```

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/shahkush589-beep/calculator.git
```

2. Navigate to the project folder

```bash
cd calculator
```

3. Open `index.html` in your preferred web browser.
